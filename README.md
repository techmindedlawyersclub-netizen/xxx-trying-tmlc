# AI Career Buddy MVP

A simple beginner-friendly web MVP for career recommendations for lawyers in the AI/tech age.

## What it does
- Asks 10 guided questions
- Groups questions into: interests, strengths, values, preferred work style
- Uses weighted scoring to rank careers
- Shows top 3 recommended careers
- Shows explanation and starter skills for each recommendation

## Run locally
Use any static server. Example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
